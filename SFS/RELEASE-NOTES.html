» What's new

The SFS2X 2.17.0 is codenamed "Flash Farewell" as it completely cuts the ties with any dependencies for the previous Flash-based AdminTool.

  • Server:
    • NEW: HTML5 AdminTool is the official and solely supported AdminTool
    • NEW: Improved Websocket performance, with reduced runtime memory footprint
macOS users only: in this release we have temporarily removed the native SFS2X.app launcher due to compatibility issues. We plan to reintegrate it with a later release.

To launch SmartFoxServer 2X:

  • Open a Terminal window
  • Move to the SFS2X installation folder
  • Run sfs2x-standalone

Example: 

  • > cd /Applications/SmartFoxServer_2X/SFS2X/
  • > ./sfs2x-standalone
